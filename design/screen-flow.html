Ship Hosted KYC

Redirect a user to a Valyd-hosted capture flow, get a signed webhook when they finish, and pull the authoritative decision. Everything you need on one page.

env VALYD_API_KEYenv VALYD_WEBHOOK_SECREThuman-only workflow_id⏱ ~10 min
1

Create a session

POST /api/v2/session with your workflow_id → returns a hosted url and session_id.

2

Redirect the user to session.url

They complete capture on Valyd’s page. The redirect-back ?status is a hint — never treat it as final.

3

Verify the webhook signature

On a terminal event, Valyd POSTs your callback. Check X-Valyd-Signature = HMAC-SHA256 of "{timestamp}.{rawBody}" before trusting it.

Node
const sig = hmacSHA256(`${ts}.${rawBody}`, SECRET);
if (!timingSafeEqual(sig, header)) return res.status(400);
4

Fetch the decision

GET /api/v2/session/{id}/decision for the authoritative result and per-check data.

# APPROVED · DECLINED · IN_REVIEW
{ "decision": "APPROVED", "checks": [ … ] }
200 OK
